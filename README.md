## 📂 Portfolio Projects

### 🌡️ Urban Heat Island Satellite Analysis — London
> Python · geospatial · rasterio · geopandas · matplotlib

Multitemporal analysis of the urban heat island effect in Greater London using Landsat 8/9 satellite imagery (2015 vs 2023). Derives Land Surface Temperature from thermal infrared data, computes LST–NDVI correlations across all 33 boroughs, and identifies the five highest-risk urban hotspots.

**Key finding:** Mean LST increased by +1.14°C across Greater London over 8 years. Tower Hamlets and Lambeth showed the sharpest warming (+1.7°C).

[→ View project](https://github.com/ao-chaos/Data_portfolio/tree/main/Urban%20Heat%20Analysis)

### 📉 What Disappears: Streaming Content Churn & Gap Analysis
> Python · pandas · matplotlib · multi-dataset analysis

Contrarian take on streaming analytics — instead of asking what's popular, this project examines what's at risk of disappearing. Analyses catalogue churn risk, content half-life, platform differentiation, and genre gaps across Netflix, Amazon Prime, and Disney+ using data from 20,000+ titles. Delivered as a market analyst brief, not a data science notebook.

**Key finding:** 24% of Netflix's library is licensed content added 6+ years after original release — the highest churn-risk segment. Disney+ and Netflix are not competing for the same audience.

[→ View project](https://github.com/ao-chaos/Data_portfolio/tree/main/Netflix%20Content%20Analysis)

### 📊 Digital Advertising Campaign Performance — SQL Analysis
> SQL · SQLite · CTEs · window functions · Python

End-to-end SQL analysis of a digital advertising account — 52 campaigns, 5 channels, 10 clients, 12 months of daily data (~£930k spend). Eight queries covering channel ROI vs industry benchmarks, campaign performance tiering, week-on-week trend analysis, campaign ranking within channels, cumulative spend tracking, and an executive monthly summary. Written to reflect real agency reporting workflows.

**Concepts demonstrated:** JOINs, CTEs, CASE WHEN, LAG, RANK, running totals, benchmark comparison.

[→ View project](https://github.com/ao-chaos/Data_portfolio/tree/main/SQL%20Ad%20Campaign%20Analysis)
